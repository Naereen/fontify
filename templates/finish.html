Use as webfont

Add the following lines to your CSS:

@font-face {
    font-family: {{ font_name }};
    src: url(fontify.ttf) format('truetype'),
         url(fontify.woff) format('woff'),
         url(fontify.otf) format('opentype');
}

/* Then use {{ font_name }} like this: */
p {
    font-family: {{ font_name }}, sans-serif;
}

Use as font in XeLaTeX

Use this template, and compile with xelatex in the good folder. See this example file (from this .tex source). 
\documentclass[a4paper,12pt]{article}
\usepackage{fontspec}
\fontspec [Path = static/css/fonts/,
    UprightFont = *-Regular,
    % ItalicFont = *-Italic   % only if you also did an Italic font
]{fontify}

\begin{document}
